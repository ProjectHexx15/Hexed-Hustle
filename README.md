<h1>Hexed Hustle</h1>

Hexed Hustle is a fast paced mobile side scroller, where you need to defeat enemies to collect candy to save Halloween! It features multiple unlockable costumes, a variety of enemies and fully functional mobile touch controls. This projecr was created for my mobile games module in college using Unity and C#.

- Mobile touch control
- 3 different enemy types
- Costumes and score saving
- Unique pixel artstyle
